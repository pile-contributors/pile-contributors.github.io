Piles

The documentation is generated using doxygen from the source code and provided on this website as an alphabetically sorted list, a list of categories and a list of tags. Each pile belongs to a single category and may be tagged with one or more tags.

{% include cmake_list.html %}

The complete reference for macros can be consulted here. A list of CMake variables that are used in the control script is also available along with a description and usage examples.

The scripts provide helpers for working with projects and with targets (executables and libraries).
