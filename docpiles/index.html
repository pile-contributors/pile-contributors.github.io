Piles Reference

Welcome to the reference documentation for piles maintained by the piles contributors.

The documentation is generated using doxygen directly from the source code and is then fine-tuned to integrate into this website.

Each pile belongs to a single category and may be tagged with one or more tags. The script that generates documentation pages extracts this information directly from .cmake file for each pile.
